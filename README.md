# 영화 예약 서비스 (movie-reservation-service)

## 영화 정보 API를 이용해 모의 예약 서비스 만들어보기

### ⚙ 프로젝트 개발 환경
- 통합개발환경 :Intellij
- JDK 버전 : JDK 17
- 스프링부트 버전 : 3.1.5
- 사용 DB : MySQL 8.0.31
- 빌드툴 : Gradle
- 관리툴 : Git, Github


### ⚒ 기술 스택
- Backend
  - Spring Boot
  - Spring Devtools
  - Lombok

- Frontend
  - JSP
  - JSTL
  - BootStrap 5.3
